Игра Жизнь
Елисеев Иван 14204 ФИТ НГУ

Спецификация:

Кнопка в меню и на панели инструментов:
1) Новая игра => Создает новую игру.
2) Открыть игрц => открывает игру из файла.
3) Сохранить => Сохраняет игру в файл.
3) Run => запускает игру.
4) Следующий шаг => следующий шаг в игре.
5) replace режим => ячейку можно сделать только в живом состояние.
6) XOR режим => Меняет ячейку на противоположную.
7) очистить => Очистить поле.
8) настройки => открываются настройки для изменения параметров.
9) значения => Показать значение каждой ячейки.
10) ? => Показать информацию о создателе.
11) выход => Выход из игры.

ограничения:
Максимальный размер поля: 100 х 100.
Максимальный размер шестиугольника: 100.
Максимальный размер границы: 5.
